页面尺寸

  1. document对象是HTML文档的根节点

    document.documentElement返回html，dom中的root节点，即<html>

    document.body返回body，dom中的body节点，即<body>

    网上很多说document.documentElement.clientHeight或者document.body.clientHeight为可见高度，在body的height(css设定)一开始就和scrollHeight相等时，即html,body不设置高度时，document.body.clientHeight是不能作为可见高度的。

    当html,body设置高度100%，document.documentElement.clientHeight == document.body.offsetHeight，都是可见高度

    document.body.clientHeight，是padding+内容高度

    document.body.offsetHeight，是border+padding+内容高度

    所有主要浏览器都支持 documentElement 属性

    页面可见区域高度：浏览器窗口页面的可是区域
    document.documentElement.clientHeight

    0

  2. 页面正文全文高，
    document.body.scrollHeight，不包括body的border
    document.body.clientHeight，html,body不设置高度时
    document.body.offsetHeight，加上body的border
    参考资料
    文档类型已申明，document.documentElement.scrollHeight为文本高度
    文档类型未申明，document.body.scrollHeight为文本高度

    0

  3. 页面滚动高度
    document.documentElement ? document.documentElement.scrollTop : document.body.scrollTop
    document.documentElement.scrollTop和document.body.scrollTop始终有一个为0，所以可以用这两个的和来求scrollTop
    参考:参考资料

    0

  4. 屏幕分辨率的宽高
    window.screen.width * window.screen.height

    0

  5. 浏览器距离屏幕右上角的距离，浏览器位置
    window.scrollLeft，window.screenTop

    0

  6. 屏幕可用工作区尺寸，就是排除了开始菜单栏的尺寸
    window.screen.availWidth，window.screen.availHeight

    0
